# VS Code Janitor

**Reclaim disk space by cleaning up old VS Code extension versions, cache files, and logs.**

VS Code accumulates gigabytes of stale data over time â€” outdated extension versions that were never removed, multiple layers of cache that rebuild automatically, and workspace storage for projects you deleted years ago. VS Code Janitor finds all of it, shows you the sizes, and lets you choose exactly what to clean.

## Commands

Open the Command Palette (`Ctrl+Shift+P`) and search for:

- **VS Code Janitor: Cleanâ€¦** â€” scan all installations, select what to clean, and run. Shows sizes for each item. Low-risk items display a warning before proceeding.
- **VS Code Janitor: Show Disk Usage** â€” scan without deleting. Outputs a full report to the Output panel.

## Supported Installations

| Installation type | Detection |
|---|---|
| Standard (Windows) | `%APPDATA%\Code\` |
| Standard (macOS) | `~/Library/Application Support/Code/` |
| Standard (Linux) | `~/.config/Code/` |
| Portable | `VSCODE_PORTABLE` env var (set automatically by VS Code) |
| Portable (manual) | `vscodeJanitor.portableDataPath` setting |
| VS Code Insiders | Opt-in via `vscodeJanitor.includeInsiders` |

When you have both a standard and a portable installation, VS Code Janitor detects and offers to clean both.
